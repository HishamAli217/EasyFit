EasyFit
===================================

.. .. include:: ../README.md
..    :parser: myst_parser.sphinx_


**EasyFit** is a python package that helps you fit different machine learning models with few lines of code.
You can easily get results for `scikit-learn`_, `xgboost`_ or your own model, with a singel object. 

.. _scikit-learn: https://scikit-learn.org/
.. _xgboost: https://xgboost.ai/

Installation:
-------------
::

   pip install easyfit
